Contexte du Projet

Money Manager est une application desktop complète développée pour la gestion personnelle des finances. L'objectif était de créer un outil intuitif permettant aux utilisateurs de suivre leurs revenus, dépenses et épargnes avec une interface moderne et sécurisée.

L'application utilise WPF (Windows Presentation Foundation) pour une interface utilisateur riche et une architecture MVVM pour une séparation claire entre la logique métier et l'interface.

Défis & Solutions

Les obstacles techniques surmontés

01

Sécurité des Données

Le défi principal était d'implémenter un système de sécurité robuste pour protéger les données financières sensibles des utilisateurs.

Solution : Implémentation du chiffrement AES-256, hachage bcrypt pour les mots de passe et validation stricte des entrées utilisateur.
02

Performance de la Base de Données

Optimiser les requêtes MySQL pour gérer de grandes quantités de transactions sans impact sur les performances.

Solution : Création d'index optimisés, utilisation de requêtes préparées et implémentation de la pagination pour les grandes listes.
03

Architecture MVVM

Implémenter correctement le pattern MVVM pour maintenir une séparation claire entre la logique métier et l'interface utilisateur.

Solution : Utilisation de INotifyPropertyChanged, Command pattern et Data Binding bidirectionnel pour une architecture propre et maintenable.
